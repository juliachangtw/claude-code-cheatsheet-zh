Claude Code v2.1.112 更新日期：2026 年 4 月 17 日
📋 近期更新 ✕
  • Claude Opus 4.7 新增 xhigh effort 等級 — 介於 high 與 max 之間，可透過 /effort、--effort 與模型選擇器使用；/effort 不帶參數時會開啟可用方向鍵調整的互動式滑桿NEW
  • /ultrareview [PR#] — 以多 Agent 平行分析進行雲端程式碼全面審查NEW
  • /theme 新增「Auto（跟隨終端機）」選項，自動對應終端機的深／淺色模式NEW
  • /less-permission-prompts Skill — 掃描對話並為 .claude/settings.json 提出優先權限白名單建議NEW
  • Plan 檔案改以 prompt 命名（例如 fix-auth-race-snug-otter.md），取代隨機字串NEW
  • Ctrl+U 清空整個輸入緩衝區、Ctrl+Y 還原；逐字稿檢視器底部新增 [（傾印至 scrollback）與 v（在編輯器開啟）NEW
  • Auto 模式不再需要 --enable-auto-mode；Max 訂閱者使用 Opus 4.7 即可啟用NEW
  • Windows PowerShell 工具開始推送 — 以 CLAUDE_CODE_USE_POWERSHELL_TOOL 加入；Linux/macOS 設 CLAUDE_CODE_USE_POWERSHELL_TOOL=1 啟用（需 PATH 有 pwsh）NEW
  • /tui fullscreen 在同一對話中啟用無閃爍渲染；/tui 指令與 tui 設定為新增項 (v2.1.110)
